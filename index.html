HTML Tag Reference

HTML Tags

Tag Name Description
<!DOCTYPE html> <!DOCTYPE> declaration A declaration that sends information to the browser about what document type to expect.
<a> <a> tag Defines a hyperlink, which is used to link from one page to another.
<article> <article> tag Specifies independent, self-contained content.
<aside> <aside> tag Defines some content aside from the content it is placed in, and should be indirectly related to the surrounding content.
<audio> <audio> tag Used to embed sound content into a document, such as music or other audio streams, and contains one or more <source> tags with different audio sources, which the browser will choose the first source it supports
<blockquote> <blockquote> tag Specifies a section that is quoted from another source; which the browser usually indents.
<body> <body> tag Defines the document's body, which contains all elements of a document body, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
<br> <br> tag inserts a single line break, which is an empty tag and has no end tag.
<button> <button> tag Defines a clickable button, and can have text and tags implemented (such as <i>, <b>, <strong>, <br>, <img>, etc.) that's not possible with the <input> element.
<datalist> <datalist> tag Specifies a list of pre-defined options for <input> element, that is used to provide an "autocomplete" feature for <input> elements, in which you'll see a dropdown list of pre-defined options as they input data.
<div> <div> tag Defines a division or a section in an HTML document, and can be used as a container for HTML elements which can be styled with CSS or manipulated with Javascript.
<embed> <embed> tag Defines a container for external resources, such as a web page, a picture, a media player, or a plug-in application.
<em> <em> tag Used to define emphasized text, and is typically displayed in italics.
<fieldset> <fieldset> tag Used to group related elements in a form, and draws a box around the related elements.
<figcaption> <figcaption> tag Defines a caption for a <figure> element.
<figure> <figure> tag Specifies self-contained content, like illustrations, diagrams, photos, code listings, etc.
<footer> <footer> tag Defines a footer for a document or section, and typically contains: authorship information, copyright information, contact information, sitemap, back to top links, and related documents.
<form> <form> tag Used to create am HTML form for user input, and can contain one or more of the following form elements: <input>, <textarea>, <button>, <select>, <option>, <optgroup>, <fieldset>, <label>, and <output>
<h1> - <h6> <h1> - <h6> tags Used to define HTML headings, in which the lower the level, the lower the impprtance.
<head> <head> tag A container for metadata (data on data) and is placed between the <html> tag and the <body> tag, and can contain elements such as: <title> (required in every HTML document), <style>, <base<, <link>, <meta>, <script>, and <noscript>.
<header> <header> tag Represents a container for introductory content or a set of navigational links, around typically contains: one or more heading elements, logo or icon, and authorship information.
<hr> <hr> tag Defines a thematic break in an HTML page, and is most often displayed as a horizontal rule that is used to seperate content (or define a change) in an HTML page.
<html> <html> tag Represents the root of an HTML document, and is a container for all other HTML elements.
<i> <i> tag Defines part of the text in an alternate voice or mood, and content inside is typically displayed in italics.
<iframe> <iframe> tag Specifies an inline frame, and is used to embed another document within the current HTML document.
<img> <img> tag Used to embed an image in an HTML page.
<input> <input> tag Specifies an input field where the user can input data, and can be displayed in several ways such as: <input type="button">, <input type="checkbox">, <input type="color">, <input type="date"<, <input type="datetime-local">, <input type="email">, <input type="file">, <input type="hidden">, <input type="image">, <input type="month">, <input type="number">, <input type="password">, <input type="radio">, <input type="range">, <input type="reset">, <input type="search">, <input type="submit">, <input type="tel">, <input type="text">, <input type="time">, <input type="url">, and <input type="week">.
<label> <label> tag Defines a label for several elements: <input type="checkbox">, <input type="color">, <input type="date"<, <input type="datetime-local">, <input type="email">, <input type="file">, <input type="month">, <input type="number">, <input type="password">, <input type="radio">, <input type="range">, <input type="search">, <input type="tel">, <input type="text">, <input type="time">, <input type="url">, and <input type="week">, <meter>, <progrss>, <select>, and <textarea>.
<li> <li> tag defines a list item.
<link> <link> tag Defines the relationship between the current document and an external resource, which is most commonly used to link external style sheets and is an empty element that contains only attributes.
<main> <main> tag Specifies the main content of the page, which should be unique to the document.
<meta> <meta> tag Defines metadata about html document, which is data(information) about data that always go inside the <head> element, and are typically used to specify character set, page description, keywords, author of the document, and viewport settings, and will not be displayed on the page, but is machine parsable. It's used by browsers (how to display content or reload page), search engines (keywords), and other web services.
<nav> <nav> tag Defines a set of navigational links.
<object> <object> tag Defines a container for an external resource, which can be a webpage, picture, media player, or a plug-in application.
<ol> <ol> tag Defines a ordered list, which can be numerical or alphabetical.
<option> <option> tag Defines an option in a select list, and can go inside <select>, <optgroup>, or <datalist> elements.
<p> <p> tag Defines a paragraph.
<script> <script> tag used to embed a client-side script(Javascript), and either contains scripting statements, or it points to an external script file through the src attribute.
<sectiont> <section> tag Defines a section in a document.
<select> <select> tag Used to create a dropdown list, and is most often used in a form.
<source> <source> tag Used to specify multiple media resources for media elements, such as <video>, <audio>, and <picture>.
<span> <span> tag An inline container used to mark up part of the text, or part of a document, that can easily be styled by CSS or manipulated with JavaScript using the class or id attribute.
<strong> <strong> tag tag is used to define text with strong importance. The content inside is typically displayed in bold.
<style> <style> tag Used to define style information (CSS) for a document.
<summary> <summary> tag defines a visible heading for the <details> element. The heading can be clicked to view/hide the details.
<svg> <svg> tag defines a container for SVG graphics, and has several methods for drawing paths, boxes, circles, text, and graphic images.
<table> <table> tag The wrapper element for all HTML tables.
<tbody> table body tag The set of rows containing actual table data.
<td> table data tag The table row container.
<textarea> <textarea> tag defines a multi-line text input control, is often used in a form, to collect user inputs like comments or reviews, and can hold an unlimited number of characters, and the text renders in a fixed-width font.
<tfoot> table foot tag set of rows defining the footer in a table.
<th> <th> tag defines a header cell in an HTML table.
<thead> table head tag The set of rows defining the column headers in a table.
<title> title tag defines the title of the document. The title must be text-only, and it is shown in the browser's title bar or in the page's tab.
<tr> table row tag The table row container.
<u> <u> tag represents some text that is unarticulated and styled differently from normal text, and is typically displayed with an underline
<ul> <ul> tag defines an unordered (bulleted) list.
<video> <video> tag used to embed video content in a document, such as a movie clip or other video streams, and contains one or more <source> tags with different video sources. The browser will choose the first source it supports.
